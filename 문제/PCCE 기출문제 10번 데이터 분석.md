# 데이터 분석 문제(정렬)

### 입력
1. data
2. ext
3. val_ext
4. sort_by
5. result

- 데이터(data)
1. 데이터는 2차원 배열 형태로 들어온다.
```python
data = [[1, 20300104, 100, 80], [2, 20300804, 847, 37], [3, 20300401, 10, 8]]
```
2. 2차원 배열의 첫번째 값은 인조 식별자인 번호이다.
3. 두번째 값은 제조일 이다.
4. 세번째 값은 최대 수량(maximum)이다
5. 네번째 값은 현재 수량(remain)이다.

- ext : 어떤 기준으로 값을 가져올지(String)
1. "code", "date", "maximum", "remain" 중 하나의 값을 가진다.
- val_ext : ext의 기준값 (val_ext 보다 작은 값만 가져온다.)
- sort_by : 정렬 기준값(오름차순 정렬)
1.  "code", "date", "maximum", "remain" 중 하나의 값을 가진다.
- result : 결과 (2차원 배열)
